# 傲慢测试 prompt

## 横版

```text
想象你去参加了下一届世界渲染大赛。这一届的主题在最后告诉你。请你带回来 9 张 横版 的参赛作品截图，把它们拼到一张大图上。九张图来自九个不同创作者，对同一个主题做出完全不同的想象。每一张都必须像真实参赛作品的最终渲染截图，不是概念草图，不是 moodboard，不是插画拼贴。

不要文字，不要标题，不要编号，不要 UI，不要水印。九宫格之间用很细的黑色间隔分开，方便后续切割。

对于每一个 idea，你都不要选择你最开始想选的那个，而是要选择第二个想到的。

这一届的主题是：傲慢
```
