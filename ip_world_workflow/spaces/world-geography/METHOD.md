# Geography Method — Places That Locate Each Other

## 1. Build the spatial model before writing any card

From the source (maps, travel chapters, wiki geography pages), sketch the world's spatial skeleton: regions, the routes characters actually travel, chokepoints, frontiers. Every location card will position itself inside this skeleton. Save the sketch to the run dir as evidence — the audit can check "X is north of Y" claims against it.

## 2. Location card anchors (per card)

- Relative position first: "three days' ride north of the capital, last stop before the frontier" — never bare coordinates or nothing.
- One visual signature a generator can hold onto (the red bridge, the terraced lights, the ash-grey harbor).
- Control & contest: current holder, challengers, why it matters to them.
- Marks of history: what happened here that a visitor could still see.
- Scene affordances: the 2-3 scene types this place naturally hosts.

## 3. Cross-linking

Cards reference each other by name (the route, the rival port, the capital it answers to). A location card with zero outbound references is an island — fix it or justify it. Character cards' relationship sections should agree with location control claims (cross-check with the character batch).

## 4. Board zoning as reading path

- Far-view test: at full zoom-out, can you tell in 5 seconds what world this is, who matters, and where things happen? 
- Mirror canonical geography when it exists (postal-code ordering, map-quadrant placement — the delight is in the details fans recognize).
- Long-tail locations group into their region's zone; never let 12 low-signal hamlets dilute the first viewport.
- Close-view test: within a zone, cards are browsable in a meaningful order (route order, hierarchy order), not upload order.

## 5. Self-check

- Could a fan redraw a rough map of the world from these cards alone?
- Does every geographic claim trace to source? (film-vs-book map divergence is a known trap — declare the canon line)
- Does the board zoning honor the Delight Plan's geography promises?
